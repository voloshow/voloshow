## About Voloshow

Voloshow is built for creators, marketers, ecommerce operators, designers, and product teams who need a complete visual workflow instead of a single prompt-to-image result.

The platform starts from a prompt or uploaded image, then continues into editing, ecommerce product visuals, asset preparation, image-to-video workflows, public galleries, prompt reuse, and remixable creative examples.

```text
Generate -> Edit -> Productize -> Animate -> Publish -> Remix
```

## Core Tools

Voloshow brings together AI generation and practical image tools in one workspace:

- **AI Image Generator** - Create original images from text prompts for concepts, campaigns, product shots, posters, social visuals, and creative drafts.
- **Image-to-Image Workflows** - Upload a reference image and guide edits, variations, transformations, or creative reinterpretations with prompts.
- **Image Tools Hub** - Continue from any uploaded or generated image with focused editing utilities.
- **Background Remover** - Remove image backgrounds for clean subject extraction and transparent assets.
- **Transparent Background** - Prepare cutout-style PNG assets for design, product graphics, and compositing.
- **White Background** - Create clean white-background visuals for ecommerce listings and marketplace-style packshots.
- **AI Background Generator** - Place subjects into styled scenes such as studio, desk, lifestyle, seasonal, luxury, or outdoor environments.
- **Image Upscaler / Enhancer** - Improve resolution and perceived quality for finished assets.
- **Image Resizer** - Export images for common web, social, ecommerce, and campaign sizes.
- **Image Compressor** - Reduce file size for publishing, upload limits, and site performance.
- **Image Converter** - Convert between formats such as PNG, JPG, and WebP.
- **Image to Video** - Turn still visuals into motion-oriented creative assets with video workflows.

## Ecommerce & Product Photo Workflows

Voloshow is expanding a focused ecommerce workflow around **AI Product Photo Generator** and **AI Ecommerce Studio**.

The goal is to help teams turn ordinary product images into usable marketplace and campaign assets:

- **White packshots** for marketplace main images.
- **Transparent PNG assets** for ads, banners, and composition.
- **Lifestyle scenes** such as minimal desk, kitchen counter, bathroom shelf, outdoor natural, holiday gift, and luxury studio.
- **Social ad creatives** for vertical and square placements.
- **Platform presets** for Amazon, Shopify, Etsy, TikTok Shop, Instagram, Xiaohongshu, and Pinterest.
- **Next actions** after generation: download, upscale, resize, compress, convert, generate similar, remix, and save to assets.

## Platform Direction

Voloshow is a production web platform with three major surfaces:

- **Product Website** - A Next.js frontend for public pages, generation workflows, image tools, gallery, pricing, and localized content.
- **Product API** - Java / Spring Boot services for generation, media, credits, users, payments, public works, prompts, tags, and operational workflows.
- **Admin Console** - A Vue 3 + Vite management interface for configuration, operations, and product administration.

The platform is designed around durable creative workflows: generate media, store results, connect them to public discovery pages, and let users continue from every useful result.

## Tech Notes

- Frontend: Next.js, React, TypeScript, Tailwind-style utility CSS, next-intl localization.
- Backend: Java 17, Spring Boot, modular product services, media and generation APIs.
- Admin: Vue 3, Vite, TypeScript, Element Plus, Pinia.
- Data and operations: PostgreSQL-oriented product data, Redis-backed runtime support, SQL migrations, Docker deployment, and production release scripts.
- Quality: Playwright checks, i18n verification, SEO smoke tests, sitemap validation, and production post-check tooling.
